# spec

本目录存放“计算资源动态调度与防爆保护”主线的设计文档。

核心文件：
1. `architecture.md`：系统架构、阈值分级、防爆策略。
2. `data_model.md`：资源快照、任务定义、调度配置、指标模型。
3. `algorithm_pseudocode.md`：调度主循环与关键决策伪代码。
4. `beginner_guide.md`：零基础可读的人话版说明。
5. `scheduler_config.example.json`：调度配置示例。

说明：
1. 旧的“记忆聚类/专利写作”相关文档已不再作为项目主线。
2. 当前实现以本目录文档为唯一设计基准。

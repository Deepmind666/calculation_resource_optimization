# 附图审校清单（R45）

## 1. 一致性检查

1. 图号与说明书引用一致（图1-图4）。
2. 图题与图内标题一致。
3. 图中术语与权利要求/说明书术语一致。
4. 图中参考标号与标记说明表一致。

## 2. 图1 审校点

1. 模块名称是否与系统权利要求一致。
2. 箭头方向是否体现闭环（监控->判定->准入/回收->执行->审计->调优）。
3. 是否存在旧方向术语残留。

## 3. 图2 审校点

1. 是否存在步骤起止标识。
2. 紧急分支与非紧急分支是否完整。
3. 是否突出“同周期累计投影”和“双目标停止”。

## 4. 图3 审校点

1. 数据来源路径是否标注。
2. A/B/C 三组定义是否与文档一致。
3. 英文标注是否需要中文化。

## 5. 图4 审校点

1. A1-A5 轮次含义是否清楚。
2. 调优动作与结果是否对应。
3. 是否突出“按失败原因分支调节”。

## 6. 提交前结论模板

1. 图1：通过/需修改（原因）  
2. 图2：通过/需修改（原因）  
3. 图3：通过/需修改（原因）  
4. 图4：通过/需修改（原因）  
5. 总结：是否允许进入提交包。

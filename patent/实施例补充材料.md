# 实施例补充材料（初稿）

## 1. 专利写作场景输入样例
```json
{
  "agents": ["search_agent", "writer_agent", "verification_agent"],
  "fragments": [
    "步骤1：提取现有技术要点，参数A=0.4",
    "步骤2：形成差异化特征，参数A=0.5",
    "风险：若证据不足需回溯原始文献"
  ]
}
```

## 2. 客服场景输入样例
```json
{
  "agents": ["service_agent", "policy_agent", "qa_agent"],
  "fragments": [
    "决定：按政策版本v3.2执行退款流程",
    "TODO：补充用户身份核验",
    "冲突：规则A与规则B对退款时限不一致"
  ]
}
```

## 3. 科研助手场景输入样例
```json
{
  "agents": ["search_agent", "analysis_agent", "writer_agent"],
  "fragments": [
    "实验结果：方法X在数据集Y上准确率提升2.1%",
    "参数范围：lr=1e-4~5e-4",
    "引用：arXiv:2310.08560"
  ]
}
```

## 4. 输出样例字段
1. `consensus_summary`
2. `disagreements`
3. `evidence_pointers`
4. `slot_coverage`
5. `compression_report`

## 5. 复现命令
```powershell
cd prototype
python main.py
python run_experiments.py
python -m unittest discover -s tests -p "test_*.py"
cd ..
```
